# delete-dialog-autofill-extension

Auto-fills the confirm input fields in delete dialogs on platforms like Vercel, GitHub, Render, Netlify, Railway, and more.

<img width="800" height="450" alt="delete-dialog-autofill-extension" src="https://github.com/user-attachments/assets/b9a4875a-33dc-4d1b-8421-8de0dc5aa5cb" />

## Installation

```bash
git clone https://github.com/sarozpradhan64/delete-dialog-autofill-extension.git
```

**Chrome:** go to `chrome://extensions`  
**Edge:** go to `edge://extensions`

1. Enable **Developer mode**
2. Click **Load unpacked**
3. Select the cloned folder
